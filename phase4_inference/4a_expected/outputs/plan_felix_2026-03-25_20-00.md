# Phase 4a Plan -- Expected Results

**Session:** Felix | **Date:** 2026-03-25 | **Phase:** 4a (Inference -- Expected)

---

## Overview

Produce expected results on MC pseudo-data for the primary Lund jet plane density.
MC reco serves as pseudo-data. Apply bin-by-bin and IBU corrections, validate
with split-sample closure and stress tests, evaluate all committed systematics,
build covariance matrices, and produce machine-readable JSON outputs.

## Scripts to write

### 01_split_sample_closure.py
- Split 40 MC files into two halves (A: files 0-19, B: files 20-39, fixed seed)
- Process each half: reco + genBefore Lund planes (reuse Phase 3 pipeline)
- Derive correction factors from half A: C_A = N_genBefore_A / N_reco_A
- Apply C_A to half B reco -> corrected half B
- Compare corrected half B to genBefore half B
- Compute chi2/ndf with Poisson errors, p-value
- Must pass p > 0.05
- Repeat for IBU method

### 02_response_matrix.py
- Build 2D -> 2D response matrix from event-matched reco/gen splittings
- Process MC files: for each event, record (reco Lund bins, gen Lund bins)
- Construct R[gen_flat, reco_flat] migration matrix
- Normalize rows to get migration probabilities
- Compute true diagonal fraction

### 03_ibu_unfolding.py
- Implement 2D iterative Bayesian unfolding
- Use response matrix from 02
- Use efficiency map from Phase 3
- Apply to MC reco (pseudo-data)
- Iterate 4 times (optimize via closure)
- Compare IBU result to genBefore truth

### 04_stress_tests.py
- Reweight MC truth by tilts: 5%, 10%, 20%, 50%
- Independent tilts in ln(k_T), ln(1/Delta_theta), plus 2D correlated
- Functional form: w(x) = 1 + epsilon * (x - x_mean) / (x_max - x_min)
- Apply reweighting at gen-level, propagate to reco via matched events
- Correct reweighted reco, compare to reweighted truth
- For both bin-by-bin and IBU methods

### 05_systematics.py
- Evaluate all committed systematic sources on MC pseudo-data
- Each produces bin-dependent shift vectors (not flat percentages)
- Sources:
  1. Tracking efficiency: drop 1% of tracks randomly
  2. Track momentum resolution: smear momenta by +/-10% of resolution
  3. Angular resolution: smear angles by +/-1 mrad
  4. Track selection cuts: vary p (150/250 MeV), |d0| (1.5/2.5 cm)
  5. Event selection cuts: vary thrust (0.6/0.8), N_ch_min (4/6)
  6. MC model dependence: reweight gen-level to tilted shapes
  7. Unfolding method: bin-by-bin vs IBU difference
  8. Heavy flavour composition: reweight b-fraction by +/-2%
  9. Thrust-axis resolution: smear thrust axis
  10. ISR modelling: use gen-level ISR/non-ISR particles

### 06_bootstrap_covariance.py
- Bootstrap N=500 event-level replicas
- For each replica: resample events with replacement, recompute full chain
- Compute sample covariance of corrected distributions -> statistical covariance
- Build per-systematic covariance: outer product of shift vectors
- Sum stat + all syst -> total covariance
- Verify PSD, condition number < 10^10

### 07_expected_results.py
- Produce corrected Lund plane from MC pseudo-data (both methods)
- Generate all figures
- Write machine-readable JSON outputs

### 08_make_figures.py
- All Phase 4a figures (see deliverable list)

## Compute estimate
- Response matrix: ~20 min (40 MC files, event-level matching)
- Split-sample closure: ~10 min (20 files each half)
- Stress tests: ~15 min (multiple reweightings)
- Systematics: ~30 min (10 sources, each requires reprocessing)
- Bootstrap: ~60 min (500 replicas -- parallelize heavily)
- Figures: ~5 min
Total: ~2-3 hours with parallelization

## Strategy: combine scripts for efficiency
Given compute constraints, combine related steps into fewer, larger scripts:
- Script 01: Response matrix + IBU + split-sample closure + stress tests (all validation)
- Script 02: All systematics (loop over sources)
- Script 03: Bootstrap covariance
- Script 04: Expected results + JSON + figures
