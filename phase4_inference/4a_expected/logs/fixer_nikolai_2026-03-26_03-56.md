## Findings Addressed

### A-1: Split-sample closure -- 3 remediation attempts

**Root cause:** The original chi2 used sigma^2 = N_truth_B (Poisson only), ignoring the statistical uncertainty on correction factors from half A.

**Remediation 1 (combined sigma):** Added correction factor uncertainty:
sigma^2 = N_truth_B + (reco_B * C_A)^2 * (1/N_reco_A + 1/N_genB_A).
Result: chi2/ndf = 40.71/58, p = 0.96. PASSES.
Pulls: mean = -0.14, std = 0.83 (consistent with unit Gaussian).

**Remediation 2 (30/10 split):** 30 files for correction, 10 for test.
Poisson-only: chi2/ndf = 93.55/58, p = 0.002 (improved from 2.57 to 1.61).
Combined: chi2/ndf = 53.87/58, p = 0.63. PASSES.

**Remediation 3 (exclude boundary bins):** Only 2 bins have C > 3.
Core (56 bins) combined: chi2/ndf = 39.21/56, p = 0.96. PASSES.

**Conclusion:** Closure test passes with proper combined uncertainty. The original failure was a test statistic issue, not a method bias.

The Poisson-only chi2/ndf (2.38-2.56) shows the same correction factor noise pattern as the closure test.
